# Assignments

This repository contains coding assignments written in C, C++, and Java.  
Each folder includes different programs based on basic programming concepts, logic building, and problem solving.

## Folder Structure

- **C/** → Contains all C language assignments  
  Examples: loops, functions, arrays, number programs, pattern printing, etc.

- **C++/** → Contains all C++ language assignments  
  Examples: classes and objects, constructors, inheritance, function overloading, file handling, etc.

- **Java/** → Contains all Java language assignments  
  Examples: basic programs, OOP concepts, exception handling, and simple applications.

## How to Use
1. Open the folder of your desired language.
2. Choose the assignment file you want to run.
3. Compile and execute it using your preferred compiler or IDE.

Example:
```bash
# For C
gcc program.c -o output
./output

# For C++
g++ program.cpp -o output
./output

# For Java
javac Program.java
java Program
